什麼是mongo

MongoDB是NoSQL的資料庫，以文件儲存資料，一般資料庫開Table須定義欄位(大小、型別、名稱等)，但是Collection完全不事先定義欄位，每筆document可以有不等數量的欄位

本文將會知道：

  • 使用docker-compose 快速啟動mongodb

MongoDB RDBMS 意思
db DB 資料庫
collection Table 表格
document record 一筆紀錄
